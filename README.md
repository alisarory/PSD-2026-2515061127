# Vector Operations Program

## Description

This program performs basic operations on a vector such as adding, deleting, searching, and calculating values like sum, maximum, and minimum.

## Features

* Add element
* Show vector
* Sum elements
* Search element
* Delete element
* Find max and min

## Output

The program displays a menu and allows the user to choose different operations on the vector.
